## Sobre este repositório

Cada pasta é um projeto independente com seu próprio README, diagramas de arquitetura e arquivos JSON prontos para importar no n8n.

Todos os workflows são validados antes de serem publicados — zero erros críticos na data de publicação.

---

## Como usar os workflows

1. Baixe o JSON do projeto que te interessa
2. No n8n, vá em **Settings → Import Workflow**
3. Selecione o arquivo e confirme
4. Configure as credenciais indicadas no README do projeto
5. Ative o workflow principal
